# mcp-screenshot

FastMCP 3.x screenshot server for macOS workflows.

## Tools

- `screenshot_smart_enhanced`: Query-aware smart capture with optional auto-zoom.
- `screenshot_smart`: Context-driven smart capture.
- `screenshot_full`: Full-screen capture (high-risk confirmation required).
- `screenshot_active_window`: Active window capture.
- `screenshot_region`: Region capture.
- `list_windows`: Enumerate windows.
- `activate_window`: Focus a specific window by ID.
- `screenshot_window`: Capture window by ID (high-risk confirmation required).
- `check_permissions`: Check Screen Recording and Accessibility status on macOS.

## Security

This server can capture sensitive on-screen data. Use it only when needed.
