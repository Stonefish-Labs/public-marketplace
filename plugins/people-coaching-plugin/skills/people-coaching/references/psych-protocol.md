# Psych Protocol Variants

Use only with guardrails and explicit user opt-in.

## Variants

- Omega: highest intensity identity challenge.
- Rebirth: staged reflective arc with lower force.
- Catalyst: compact 5-prompt version.
- Dual-Track: paired piercing and integrative prompts.

## Recommended Default

Start with Rebirth or Dual-Track unless user explicitly requests higher intensity.

## Rebirth Stages

1. Soft unveiling
2. Fracture point
3. Excavation
4. Reformation

## Usage Rules

- run one prompt at a time
- avoid certainty claims
- label inferences explicitly
- pause on distress signals
