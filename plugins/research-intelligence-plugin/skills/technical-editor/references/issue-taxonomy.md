# Technical Editorial Issue Taxonomy

Allowed categories:

- Unnecessary verbosity
- Redundancy
- Vague language
- Implied context / assumed knowledge
- Weak verbs
- Nominalization
- Passive voice (when harmful)
- Temporal ambiguity
- Emotional or subjective language
- Sentence does not advance meaning
- Over-qualification
- Inconsistent tense
- Inconsistent terminology
- Structural issue
- List misuse
- Scope creep
- Parenthetical misuse

## Output Format

For each issue:

---
Quoted text:
"<exact quote>"

Issue category:
<one or more categories>

What's wrong:
<concise explanation>

Recommendation:
REMOVE | REWORD | KEEP

Suggested revision (only if REWORD):
"<improved version>"
---

At end:

- total issues found
- overall assessment (1-2 sentences)
