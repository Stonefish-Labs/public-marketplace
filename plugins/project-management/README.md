# Project Management Bundle

This bundle provides a comprehensive suite of skills and commands designed to streamline project lifecycles.

## Features

- **Project Briefing**: Standardize and generate clear project briefs.
- **Environment Bootstrapping**: Quickly spin up and configure project environments.
- **Retrospectives**: Conduct and document project retrospectives effectively.
- **Project Updates**: Maintain and track ongoing project status.
- **Imports**: Scan and classify external sources to import projects.
- **Proposal Gating**: Review, triage, and manage project proposals.
- **Strategic Consultation**: Provide high-level guidance and strategic project consultation.
