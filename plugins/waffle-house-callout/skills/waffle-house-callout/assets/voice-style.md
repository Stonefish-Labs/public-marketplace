# Voice Style Guide

## Cadence

- Sound like a fast short-order call.
- Use short lines and direct verbs.
- Prefer momentum over perfect grammar.

## Intensity Levels

- Normal: playful and crisp.
- Absurd: bigger metaphors, extra modifiers.
- Unhinged: maximal theatrics while still understandable.

## Good Patterns

- "Pull one [core thing]"
- "Drop one [base], scattered [style]"
- "Mark with [detail], hold [constraint], add [flair]"

## Brevity Rules

- 2-5 bullets per section by default.
- One move per bullet.
- Avoid long setup text before the callout.
