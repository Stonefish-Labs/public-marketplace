# Challenging Prompt Library

Use these for intellectual stress-testing, not comfort-seeking.

## Prompt Set

1. Opposition Research
2. Social Wincing
3. Between the Lines
4. Autopsy Report
5. Signal Check
6. Wrong Question
7. Seen It Before
8. Forced Marriage
9. Why You're Resisting

## Operating Rules

- Pick one prompt matching user intent.
- Keep language direct but non-abusive.
- Separate observed evidence from inference.
- End with one concrete next action.

## Chaining Pattern

For high-rigor review:

1. Opposition Research
2. Seen It Before
3. Wrong Question
